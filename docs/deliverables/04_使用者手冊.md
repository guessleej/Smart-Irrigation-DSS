## 1. 系統簡介

### 1.1 系統概述

農業水資源智慧決策支援系統（Smart Irrigation Decision Support System, SIDSS）是一套整合多元水資源與氣象資料的智慧平台，提供視覺化分析與風險評估功能，協助灌溉管理者做出最佳決策。

### 1.2 主要功能

本系統提供以下主要功能模組：

| 功能模組 | 說明 |
|---------|------|
| 決策儀表板 | 整合即時水情統計、降雨情勢、水庫豐枯分析 |
| 水庫即時監測 | 顯示全台 68 座水庫即時水情資料 |
| 缺水風險評估 | 評估各灌區灌溉缺水風險等級 |
| 動態配水模擬 | 模擬不同情境下的最佳配水方案 |
| 3D GIS 視覺化 | 地圖互動展示灌溉系統設施分布 |
| 灌區管理 | 管理灌區基本資料與設施清單 |

### 1.3 系統需求

| 項目 | 最低需求 |
|------|---------|
| 瀏覽器 | Chrome 90+、Firefox 90+、Edge 90+、Safari 14+ |
| 螢幕解析度 | 1280 x 720 以上 |
| 網路連線 | 穩定的網際網路連線 |

## 2. 系統登入

### 2.1 登入步驟

使用者可透過以下步驟登入系統：

**步驟一：** 開啟瀏覽器，輸入系統網址。

**步驟二：** 點擊首頁右上角的「進入系統」按鈕。

**步驟三：** 系統將導向 OAuth 登入頁面，請使用您的帳號密碼登入。

**步驟四：** 登入成功後，系統將自動導向決策儀表板頁面。

### 2.2 登出步驟

**步驟一：** 點擊頁面右上角的使用者名稱。

**步驟二：** 在下拉選單中點擊「登出」。

**步驟三：** 系統將清除登入狀態並導回首頁。

## 3. 決策儀表板

### 3.1 功能說明

決策儀表板為系統首頁，整合即時水情統計、降雨情勢、水庫豐枯分析等關鍵資訊，提供決策者快速掌握整體水資源狀況。

### 3.2 介面說明

決策儀表板包含以下區塊：

| 區塊名稱 | 說明 |
|---------|------|
| 統計卡片區 | 顯示水庫總數、平均蓄水率、警戒水庫數、灌區統計 |
| 水庫蓄水率趨勢圖 | 顯示近 7 日水庫蓄水率變化趨勢 |
| 降雨量趨勢圖 | 顯示近 7 日降雨量變化趨勢 |
| 風險評估摘要 | 顯示最新風險評估結果 |

### 3.3 操作說明

**查看統計資料：** 統計卡片區會自動顯示最新的水情統計資料，包含監測中水庫總數、平均蓄水率、警戒水庫數量等。

**查看趨勢圖表：** 將滑鼠移至圖表上方，可顯示該時間點的詳細數值。

**初始化示範資料：** 若系統尚無資料，可點擊「初始化示範資料」按鈕，系統將自動產生示範資料供測試使用。

## 4. 水庫即時監測

### 4.1 功能說明

水庫即時監測模組串接水利署開放資料平台 API，提供全台水庫即時水情資料查詢與視覺化呈現。

### 4.2 介面說明

水庫監測頁面包含以下區塊：

| 區塊名稱 | 說明 |
|---------|------|
| 資料來源切換 | 可切換「即時 API 資料」或「資料庫資料」 |
| 搜尋列 | 輸入水庫名稱進行搜尋 |
| 統計摘要 | 顯示水庫總數、平均蓄水率、警戒水庫數 |
| 水庫清單 | 以卡片形式顯示各水庫詳細資料 |

### 4.3 操作說明

**切換資料來源：**

| 資料來源 | 說明 |
|---------|------|
| 即時 API 資料 | 從水利署 API 取得最新即時資料 |
| 資料庫資料 | 從系統資料庫取得已儲存的資料 |

點擊頁面上方的切換按鈕，可在兩種資料來源間切換。

**搜尋水庫：** 在搜尋列輸入水庫名稱（如「石門」、「翡翠」），系統將即時篩選符合條件的水庫。

**查看水庫詳細資料：** 每張水庫卡片顯示以下資訊：

| 欄位 | 說明 |
|------|------|
| 水庫名稱 | 水庫的正式名稱 |
| 蓄水量 | 目前蓄水量（萬立方公尺） |
| 蓄水率 | 目前蓄水量佔有效容量的百分比 |
| 有效容量 | 水庫的有效蓄水容量 |
| 進水量 | 每秒進水量（立方公尺/秒） |
| 出水量 | 每秒出水量（立方公尺/秒） |
| 水位 | 目前水位高度（公尺） |
| 狀態 | 正常（綠色）/ 警戒（黃色）/ 嚴重警戒（紅色） |

**同步資料至資料庫：** 點擊「同步至資料庫」按鈕，可將即時 API 資料儲存至系統資料庫。

## 5. 缺水風險評估

### 5.1 功能說明

缺水風險評估模組結合水資源供需模擬，評估各灌區之灌溉缺水風險等級，提供決策者預警資訊。

### 5.2 風險等級說明

| 風險等級 | 風險指數 | 顏色標示 | 說明 |
|---------|---------|---------|------|
| 低風險 | 0 - 25 | 綠色 | 水資源充足，無需特別措施 |
| 中風險 | 26 - 50 | 黃色 | 水資源略顯不足，建議關注 |
| 高風險 | 51 - 75 | 橘色 | 水資源不足，建議採取節水措施 |
| 極高風險 | 76 - 100 | 紅色 | 水資源嚴重不足，需緊急應變 |

### 5.3 操作說明

**查看風險評估結果：** 頁面會自動顯示各灌區的最新風險評估結果，包含風險指數、風險等級、評估因子等。

**執行風險評估：** 點擊「執行風險評估」按鈕，系統將重新計算各灌區的風險指數。

**查看評估因子：** 展開風險評估卡片，可查看影響風險指數的各項因子：

| 因子 | 說明 | 權重 |
|------|------|------|
| 水庫蓄水率 | 相關水庫的平均蓄水率 | 40% |
| 降雨充足度 | 近期降雨量與歷史平均的比較 | 30% |
| 灌溉需求指數 | 當季作物的灌溉需水量 | 30% |

**查看建議措施：** 系統會根據風險等級提供對應的建議措施。

## 6. 動態配水模擬

### 6.1 功能說明

動態配水模擬模組以桃園湖口工作站為例，模擬不同情境下的配水方案，協助決策者規劃最佳水資源配置。

### 6.2 模擬情境說明

| 情境 | 說明 |
|------|------|
| 正常情境 | 水資源供應正常，依標準配水比例分配 |
| 乾旱情境 | 水資源供應減少 30%，需調整配水優先順序 |
| 豪雨情境 | 水資源供應充足，可增加配水量 |

### 6.3 操作說明

**設定模擬參數：**

**步驟一：** 選擇模擬情境（正常/乾旱/豪雨）。

**步驟二：** 輸入可用水量（萬立方公尺）。

**步驟三：** 選擇灌區範圍。

**步驟四：** 點擊「執行模擬」按鈕。

**查看模擬結果：** 模擬完成後，系統將顯示各灌區的建議配水量、配水效率指標等資訊。

| 結果欄位 | 說明 |
|---------|------|
| 灌區名稱 | 配水目標灌區 |
| 建議配水量 | 根據模擬計算的建議配水量 |
| 配水比例 | 該灌區配水量佔總配水量的比例 |
| 滿足率 | 配水量佔灌溉需求量的比例 |

## 7. 3D GIS 視覺化

### 7.1 功能說明

3D GIS 視覺化模組整合 Google Maps API，提供灌溉系統設施與管理資訊之地圖互動展示。

### 7.2 地圖圖層說明

| 圖層 | 說明 | 圖示 |
|------|------|------|
| 水庫 | 顯示水庫位置與狀態 | 藍色水滴圖示 |
| 灌區 | 顯示灌區範圍 | 綠色區域 |
| 設施 | 顯示灌溉設施位置 | 橘色圓點 |

### 7.3 操作說明

**地圖基本操作：**

| 操作 | 方式 |
|------|------|
| 縮放 | 滑鼠滾輪或點擊 +/- 按鈕 |
| 平移 | 按住滑鼠左鍵拖曳 |
| 旋轉 | 按住 Ctrl + 滑鼠左鍵拖曳 |

**查看標記詳細資訊：** 點擊地圖上的標記（水庫、設施等），將彈出資訊視窗顯示詳細資料。

**切換圖層：** 點擊地圖右上角的圖層控制按鈕，可開啟或關閉各圖層的顯示。

## 8. 灌區管理

### 8.1 功能說明

灌區管理模組提供灌區基本資料、設施清單與作物資訊之管理功能。

### 8.2 操作說明

**查看灌區清單：** 頁面會顯示所有灌區的基本資料，包含名稱、面積、主要作物、設施數量等。

**搜尋灌區：** 在搜尋列輸入灌區名稱，系統將即時篩選符合條件的灌區。

**查看灌區詳細資料：** 點擊灌區卡片，可展開查看詳細資料與設施清單。

**管理灌區資料（限管理員）：** 具有管理員權限的使用者，可進行灌區資料的新增、修改、刪除操作。

## 9. 常見問題

### Q1：系統顯示「載入中」很久沒有回應？

**A：** 可能是網路連線不穩定或外部 API 服務暫時無法使用。請檢查網路連線，或稍後再試。若問題持續，請聯繫系統管理員。

### Q2：水庫資料顯示「暫無資料」？

**A：** 可能是外部 API 服務暫時無法使用。請嘗試切換至「資料庫資料」模式，或點擊「重新整理」按鈕。

### Q3：如何取得管理員權限？

**A：** 請聯繫系統管理員，由管理員在系統後台將您的帳號角色設定為「Admin」。

### Q4：風險評估結果多久更新一次？

**A：** 系統預設每小時自動執行一次風險評估。您也可以手動點擊「執行風險評估」按鈕立即更新。

### Q5：如何匯出資料？

**A：** 目前系統支援配水模擬結果的匯出功能。在模擬結果頁面，點擊「匯出」按鈕即可下載 CSV 格式的報表。
