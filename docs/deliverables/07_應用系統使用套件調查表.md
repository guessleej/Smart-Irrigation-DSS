建議定期執行以下指令檢查套件安全性：

```bash
pnpm audit
```

### 6.2 套件更新策略

| 更新類型 | 頻率 | 說明 |
|---------|------|------|
| 安全性更新 | 立即 | 發現安全漏洞時立即更新 |
| 次要版本更新 | 每月 | 功能更新與錯誤修復 |
| 主要版本更新 | 每季 | 需評估相容性後更新 |
